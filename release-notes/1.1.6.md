# CircuitSim 1.1.6

Date: 2026-03-15

## Highlights
- CircuitSim now checks GitHub for a newer release when a packaged build starts.
- The update prompt can open the matching release asset directly when GitHub provides a suitable download for the current platform.

## Download / Update
- Windows installer: download and run the latest `.exe` installer to install or update in place.
- Linux Debian/Ubuntu (`.deb`) install or update:
  ```sh
  sudo apt install ./circuitsim_1.1.6_amd64.deb
  ```
- Linux Fedora/RHEL/openSUSE (`.rpm`) install or update:
  ```sh
  sudo rpm -Uvh ./circuitsim-1.1.6-1.x86_64.rpm
  ```
- Linux portable (`.tar.gz`):
  ```sh
  tar -xzf CircuitSim-linux-portable-1.1.6.tar.gz
  ./CircuitSim/bin/CircuitSim
  ```
- Linux Flatpak install:
  ```sh
  flatpak install --user ./CircuitSim-linux-1.1.6.flatpak
  ```
- Linux Flatpak update:
  ```sh
  flatpak update com.blazinghotcode.CircuitSim
  ```

## Notes
- Update checks are skipped for source/dev runs that do not carry a packaged `Implementation-Version`.
