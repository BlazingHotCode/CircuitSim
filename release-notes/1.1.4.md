# CircuitSim 1.1.4

Date: 2026-03-15

## Highlights
- Windows installers now upgrade existing CircuitSim installs instead of forcing a manual uninstall first.
- Packaged Windows builds now keep the CircuitSim icon in the running app, launcher, and Start menu metadata.
- GitHub Releases can now publish Windows and Linux downloads automatically from version tags.

## Notes
- Windows Start menu shortcuts are now grouped under `BlazingHotCode`.
- GitHub Release descriptions now use the matching `release-notes/<version>.md` file.
