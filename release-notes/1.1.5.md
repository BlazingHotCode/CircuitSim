# CircuitSim 1.1.5

Date: 2026-03-15

## Highlights
- Linux releases now ship in more formats: `.deb`, `.rpm`, portable `.tar.gz`, app image, and Flatpak.
- GitHub Releases now include the versioned application jar used by the Flathub submission manifest.
- Added Flathub submission helpers and a reusable release-notes template for all package formats.

## Download / Update
- Windows installer: download and run the latest `.exe` installer to install or update in place.
- Linux Debian/Ubuntu (`.deb`) install or update:
  ```sh
  sudo apt install ./circuitsim_1.1.5_amd64.deb
  ```
- Linux Fedora/RHEL/openSUSE (`.rpm`) install or update:
  ```sh
  sudo rpm -Uvh ./circuitsim-1.1.5-1.x86_64.rpm
  ```
- Linux portable (`.tar.gz`):
  ```sh
  tar -xzf CircuitSim-linux-portable-1.1.5.tar.gz
  ./CircuitSim/bin/CircuitSim
  ```
- Linux Flatpak install:
  ```sh
  flatpak install --user ./CircuitSim-linux-1.1.5.flatpak
  ```
- Linux Flatpak update:
  ```sh
  flatpak update com.blazinghotcode.CircuitSim
  ```

## Notes
- Flathub submission files are now documented in `packaging/flatpak/FLATHUB.md`.
