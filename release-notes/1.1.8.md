# CircuitSim 1.1.8

Date: 2026-03-17

## Highlights
- Linux release artifacts are now split by architecture so `x86_64` and `arm64` downloads are labeled clearly.
- Flatpak builds now bundle their own Java runtime and use a more reliable launcher path for Linux desktop sessions.
- Short-circuit warnings now stay visible without sitting behind the wire color palette.

## Download / Update
- Windows installer: download and run the latest `.exe` installer to install or update in place.
- Linux Debian/Ubuntu (`.deb`) install or update (`amd64` example):
  ```sh
  sudo apt install ./circuitsim_1.1.8-1_amd64.deb
  ```
- Linux Fedora/RHEL/openSUSE (`.rpm`) install or update (`x86_64` example):
  ```sh
  sudo rpm -Uvh ./circuitsim-1.1.8-1.x86_64.rpm
  ```
- Linux portable (`.tar.gz`) (`x86_64` example):
  ```sh
  tar -xzf CircuitSim-linux-portable-1.1.8-x86_64.tar.gz
  ./CircuitSim/bin/CircuitSim
  ```
- Linux Flatpak install (`x86_64` example):
  ```sh
  flatpak install --user ./CircuitSim-linux-1.1.8-x86_64.flatpak
  ```
- Linux Flatpak update:
  ```sh
  flatpak update com.blazinghotcode.CircuitSim
  ```

## Notes
- Linux GitHub Release assets now ship in separate `x86_64` and `arm64` variants; choose the file that matches your CPU architecture.
- Flatpak packaging now includes a bundled Java runtime and improved X11 display detection for Swing on Linux.
