# CircuitSim 1.1.7

Date: 2026-03-15

## Highlights
- Added a top-right version badge that shows the installed version and the latest GitHub release status.
- Startup update checks now fall back more gracefully and surface clearer status details, including SSL/browser fallback behavior.
- Windows packaging now supports cleaner `-Type exe` runs and prints step-by-step progress while building.

## Download / Update
- Windows installer: download and run the latest `.exe` installer to install or update in place.
- Linux Debian/Ubuntu (`.deb`) install or update:
  ```sh
  sudo apt install ./circuitsim_1.1.7_amd64.deb
  ```
- Linux Fedora/RHEL/openSUSE (`.rpm`) install or update:
  ```sh
  sudo rpm -Uvh ./circuitsim-1.1.7-1.x86_64.rpm
  ```
- Linux portable (`.tar.gz`):
  ```sh
  tar -xzf CircuitSim-linux-portable-1.1.7.tar.gz
  ./CircuitSim/bin/CircuitSim
  ```
- Linux Flatpak install:
  ```sh
  flatpak install --user ./CircuitSim-linux-1.1.7.flatpak
  ```
- Linux Flatpak update:
  ```sh
  flatpak update com.blazinghotcode.CircuitSim
  ```

## Notes
- If Java cannot complete GitHub HTTPS requests on a machine, the version badge shows an SSL status and opens the releases page in the system browser instead.
